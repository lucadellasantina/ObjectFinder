What is ObjectFinder?

ObjectFinder is a MATLAB® app that allows you to recognize a large number of small structures, such as synapses in neuronal tissue, within a three-dimensional image volume.

ObjectFinder was originally developed for neuroscience research purposes, where it is able to detect fluorescently-labeled synapses in neuronal image stacks acquired using confocal or super-resolution microscopes.

Tuned for speed!

ObjectFinder can search hundred thosands objects in a matter of minutes by taking full advantage of your computer's CPU cores, this is achieved by implementing multi-threaded search inside the 3D volume.

For a typical workstation with an 8-core CPU this means an 8X fold faster speed compared to a classic app running on the same computer!

Multiplatform architecture

Whether your workstation relies on Microsoft Windows, macOS or Linux, ObjectFinder can run on your computer since it relies only on core Matlab functions.

Having a single codebase for all platforms makes developing new features as well as fixing bugs a much faster and enjoyable journey.

Seamlessly integrated with Bitplane Imaris®

ObjectFinder can display detected objects in Imaris® to take full advantage of Imaris advanced 3D rendering capabilities.

You can visually add, remove or filter objects by visual inspection against the original signal and export them back to ObjectFinder for further processing.

Export results to Microsoft Excel®

Analysis results can be exported to Excel spreadsheets for further elaboration or plotting using your favorite data analysis application.

Works around your schedule

Schedule ObjectFinder to start the automatic object finding process anytime within 24 hours to keep working on your other projects. Then let ObjectFinder do the work during your quiet hours, day or night. Do you have multiple images to process? Use the automation tab to allow batch processing without user supervision!

How do I install and use ObjectFinder?

  1. Download latest ObjectFinder version from the top of this page
  2. Install ObjectFinder using MATLAB's "Install App" button
  3. Start ObjectFinder from your Apps list.

The object recognition process starts with you creating a folder structure for your experiment. Let's assume your experiment folder's name is called "MyExperiment", place all your original image stacks inside the folder "MyExperiment/I/" saved as 3D .tif files (one file per channel).

Start Objectfinder and use the main toolbar to perform all the steps of the analysis:

  1. Select "MyExperiment" as the current working folder using the first home folder icon.
  2. Click "Load Images" to visualize all available images and instruct ObjectFinder on which image contains the structures to recognize. Optionally, you can also specify which a 3D binary mask to limit the search.
  3. Click "Search Objects" to start the automatic search of objects.
  4. Click "Filter Objects" to remove candidate objects that do not meet the specified criteria
  5. Click on "Inspect 3D" ff you have Imaris® installed to visually inspect and validate objects.
  6. Click "Calc. Density" to finish the detection process and calculate density statistics.

System Requirements

Software:
  • MATLAB R2016b or newer
  • Image Processing Toolbox for MATLAB
  • Parallel Computing Toolbox for MATLAB
  • Bitplane Imaris for 3D inspection (optional, tested with version 7)
Hardware (recommended):
  • Processor (CPU): Intel Xeon >2GHz
  • System Memory (RAM): 32Gb
  • Video Card (GPU): NVidia Quadro with 4Gb VRAM
